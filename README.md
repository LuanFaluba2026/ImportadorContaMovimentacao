# Conversor XML - Movimentação de Contas
Projeto desenvolvido para facilitar e automatizar a importação contábil nos sistemas Domínio e Sênior, permitindo o uso de planilhas ou arquivos XML como fonte de dados.
### Tecnologias utilizadas:
- Banco de Dados com SQLite.
- ClosedXML para leitura e geração de planilhas.
- Windows Forms / .NET
### Funcionalidades: 
- Importação e duplicação do plano de contas contábeis de ambos sistemas.
- Conversão de XML para documento .xlsx/.csv no layout correto para importação.
- Gerenciamento de contas contábeis (Adicionar, remover e consultar).
- Ediçaõ simples e rápida dos movimentos.
- Vinculo entre Conta Contábil e Fornecedor.
### Modo de Uso:
1. Cadastrar a empresa utilizando o mesmo código existente no sistema contábil.
2. Importar ou duplicar o plano de contas da empresa.
3. Importar os arquivos XML ou a planilha com os dados das notas fiscais.
4. Conferir, ajustar e processar as informações.
5. Gerar automaticamente o arquivo final na pasta escolhida, pronto para importação no sistema.
